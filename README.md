# CP-Stardew-Valley
A small Stardew Valley Content Patcher mods.

## Requirements
- SMAPI
- Content Patcher

## Installation
1. Download and install [SMAPI](https://smapi.io/).
2. Install [Content Patcher](https://www.nexusmods.com/stardewvalley/mods/1915)
3. Place this mod folder into `Stardew Valley/Mods`.

## Lewis Better
- Changes Lewis's mustache and hair to black.
- Changes his shirt color for his normal and winter outfits.
- Removes his shirt in his beach outfit.
